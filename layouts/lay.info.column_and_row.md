# Column and Row

## MainAxisAlignment

### MainAxisAlignment.start


```dart 
Row /*or Column*/ (
  mainAxisAlignment: MainAxisAlignment.start,
  children: <Widget>[
    Icon(Icons.flutter_dash, size: 80),
    Icon(Icons.flutter_dash, size: 80),
    Icon(Icons.flutter_dash, size: 80),
  ],
)
```
